# 日本语同学会

一个面向手机和桌面的日语学习 PWA，包含 50 音、日常词汇、汉字短词、JLPT N1-N5 模拟测试与语法学习。

## 功能特点

- **50音基础**：清音、浊音、半浊音、拗音练习
- **日常短词**：常用词汇读写、释义、同义词、近义词、反义词练习
- **汉字短词**：汉字读音、表记、意思练习
- **日本语测试**：JLPT N1-N5 模拟题，支持测试模式、答案学习、循序学习
- **语法学习**：动词分组等主题，支持情景练习、发音和用法说明
- **多语言界面**：简体中文、繁体中文、日语、英语
- **答案辅助**：JLPT 题目含答案解释与原文译文
- **PWA 支持**：可添加到 iOS/Android 桌面，支持 Service Worker 自动更新

> 说明：JLPT 题库是按 JLPT 题型整理/生成的模拟练习题，不是官方历年真题。

## 本地运行

需要通过 HTTP 服务器访问（ES Modules 不支持 file:// 协议）：

```bash
python -m http.server 8080
```

然后访问 http://localhost:8080

## 版本号

页面底部显示版本号，格式为：`YYYY-MM-DD.N`。本项目已配置 pre-commit hook 自动更新：

```bash
sh scripts/install_hooks.sh
```
