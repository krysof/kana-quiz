# Kana Quiz (GitHub Pages)

## 功能
- 内容/题型/50音集合（清音/浊音/半浊音）均为复选框，可多选
- 默认只练清音；浊音/半浊音默认不勾
- 每局题数可改（5~200），所有设置自动保存
- 本次训练正确率 + 每日正确率
- 错题加权抽取；勾“错题优先”权重更大
- 点击题目/选项可发音（浏览器 TTS）
- 手机适配

## 部署
1. 新建 GitHub 仓库，上传本项目全部文件
2. Settings → Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
3. 访问：
   https://<用户名>.github.io/<仓库名>/
